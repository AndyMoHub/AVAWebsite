Snow Plow Script

The results of the table when the script is run. The TotalPlowCost shows how much it would cost to plow that section of the road segment. This is also visualized in the map, where the darker reds indicate higher snow plow costs and lighter shades representing lower costs.

Storm and Sanitary Script

The results of the table when the script is run. The picture to the left shows what the geodatabase looks like after the script is run, present are the 6 feature datasets, along with the created topology rules, network dataset, and closest facility output routes. This is visualized using manual symbology to demonstrate what the closest facility outputs (Figure to the right).

Cartographic Output

In order to show the full network of both the Sanitary and Storm sewage line, a full citywide map was created. Having a visual representation of the line is necessary in understanding the scope of this project. The map also gives a small description outlining the purpose of this project along with an inset map to give a zoomed in scale of all the features highlighted.
